Jenkins is an open-source automation tool used for Continuous
Integration (CI) and Continuous Deployment (CD) in software
development. It helps developers automatically build, test, and deploy
applications, making the development process faster and more reliable.
Jenkins supports a wide range of plugins, allowing it to integrate with
various tools like Git, Docker, and Maven. It runs on a server and
provides a web-based interface to manage automation pipelines.
Key points:
 Open-source automation server
 Used for CI/CD pipelines
 Supports many plugins for integration
 Automates build, test, and deployment processes
 Improves software quality and delivery speed

CI/CD (Continuous Integration and Continuous

Deployment/Delivery)

CI/CD (Continuous Integration and Continuous Deployment/Delivery)
is a software development practice that automates the process of
building, testing, and deploying code. It helps developers frequently

integrate code changes into a shared repository and ensures that the
application is always in a deployable state.
Key points:
 Continuous Integration (CI): Automatically builds and tests code
whenever changes are made
 Continuous Delivery (CD): Ensures code is ready for deployment
at any time
 Continuous Deployment (CD): Automatically deploys code to
production without manual steps
 Reduces errors and improves code quality
 Speeds up development and release cycles
Prometheus

Prometheus is an open-source monitoring and alerting tool used to
collect and analyze system and application metrics in real time. It stores
data as time-series and helps track performance, detect issues, and
generate alerts when problems occur.
Key points:
 Open-source monitoring system
 Collects real-time metrics (CPU, memory, requests, etc.)
 Uses time-series database for data storage
 Powerful query language (PromQL)
 Supports alerting and integrates with tools like Grafana
 Widely used in cloud and DevOps environments

Grafana

Grafana is an open-source data visualization and monitoring tool used to
create interactive dashboards from various data sources. It helps users
visualize metrics, logs, and system performance in real time, making it
easier to monitor and analyze applications and infrastructure.
Key points:
 Open-source visualization tool
 Creates interactive and customizable dashboards

 Connects with data sources like Prometheus, MySQL, and
Elasticsearch
 Displays real-time metrics and logs
 Supports alerts and notifications
 Widely used in DevOps and cloud monitoring
