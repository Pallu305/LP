# 🔷 2. Theory

## 📌 What is CSP (Constraint Satisfaction Problem)?

A CSP consists of:

* **Variables** → positions of queens
* **Domain** → possible columns (0 to N-1)
* **Constraints** → no two queens attack each other

---

## 📌 N-Queens Problem

Place **N queens on an N×N chessboard** such that:

* No two queens are in the same row
* No two queens are in the same column
* No two queens are on the same diagonal

---

## 📌 Representation

We use a **1D array**:

```
board[i] = column position of queen in row i
```

Example:

```
[1, 3, 0, 2]
```

Means:

* Row 0 → Column 1
* Row 1 → Column 3
* Row 2 → Column 0
* Row 3 → Column 2

---

# 🔷 3. Methods Used

---

## 🔹 A) Backtracking

### ✔ Idea:

* Try placing queen row by row
* If conflict occurs → backtrack
* Continue until solution is found

### ✔ Time Complexity:

* Worst case: **O(N!)**

---

## 🔹 B) Branch and Bound

### ✔ Idea:

* Use extra arrays to **avoid unnecessary checks**
* Track:

  * Columns
  * Left diagonals
  * Right diagonals

### ✔ Advantage:

* Faster than backtracking (prunes search space)

# 🔷 4. Python Implementation (Corrected & Complete)

class NQueens:

    # ---------------- BACKTRACKING ----------------
    def solve_backtracking(self, n):
        print("\nSolution using Backtracking:")
        board = [-1] * n

        def is_safe(row, col):
            for i in range(row):
                if board[i] == col or abs(board[i] - col) == abs(i - row):
                    return False
            return True

        def solve(row):
            if row == n:
                print("Board:", board)
                return True

            for col in range(n):
                if is_safe(row, col):
                    board[row] = col

                    if solve(row + 1):
                        return True

                    board[row] = -1  # Backtrack

            return False

        if not solve(0):
            print("No solution exists.")

    # ---------------- BRANCH AND BOUND ----------------
    def solve_branch_bound(self, n):
        print("\nSolution using Branch and Bound:")
        board = [-1] * n

        columns = [False] * n
        left_diag = [False] * (2 * n)
        right_diag = [False] * (2 * n)

        def solve(row):
            if row == n:
                print("Board:", board)
                return True

            for col in range(n):
                if (not columns[col] and
                    not left_diag[row - col + n] and
                    not right_diag[row + col]):

                    board[row] = col
                    columns[col] = True
                    left_diag[row - col + n] = True
                    right_diag[row + col] = True

                    if solve(row + 1):
                        return True

                    # Backtrack
                    columns[col] = False
                    left_diag[row - col + n] = False
                    right_diag[row + col] = False

            return False

        if not solve(0):
            print("No solution exists.")

n = int(input("Enter value of N: "))

solver = NQueens()
solver.solve_backtracking(n)
solver.solve_branch_bound(n)
