def job_sequencing(jobs):
    # Sort by profit (3rd element) in descending order
    jobs.sort(key=lambda x: x[2], reverse=True)
    
    # Create slots based on maximum deadline
    slots = [None] * max(j[1] for j in jobs)
    order = []
    total_profit = 0

    # Assign jobs to slots
    for j_id, d, p in jobs:
        for i in range(d - 1, -1, -1):
            if slots[i] is None:
                slots[i] = j_id
                total_profit += p
                order.append(j_id)
                break

    return order, total_profit

# 🔹 1. Concept: Job Sequencing Problem

You are given:

* `n` jobs
* Each job has:

  * **Deadline (d)** → last time slot to finish
  * **Profit (p)** → earned if completed on time

👉 Goal:
**Maximize total profit** by scheduling jobs within deadlines.

---

# 🔹 2. Key Idea (Greedy Strategy)

Use **Greedy Approach**:

1. Sort jobs in **descending order of profit**
2. Pick highest profit job first
3. Assign it to the **latest possible free slot before its deadline**

👉 Why latest slot?
To leave earlier slots free for other jobs.

---

# 🔹 3. Algorithm Steps

1. Sort jobs by profit (descending)
2. Find **maximum deadline**
3. Create `slots[]` array of that size
4. For each job:

   * Try to place it from `(deadline - 1)` to `0`
   * Assign if slot is empty
